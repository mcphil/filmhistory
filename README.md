# Die Geschichte des Bewegtbildes

Eine interaktive Timeline-Website zur Geschichte des Films von den Ursprüngen bis 2026.

## Über das Projekt

Diese Website erzählt die Geschichte des Bewegtbildes – von der Camera Obscura bis zu KI-generierten Videos (Sora, 2024). Der rote Faden: Wie Technik und Filmsprache sich gegenseitig herausgefordert, befreit und manchmal auch gefesselt haben.

**Inhalte:**

- Interaktive Timeline mit 10 Epochen (Vorfilm bis KI-Ära)
- ~70 Timeline-Einträge mit aufklappbaren Texten und Zitaten
- Überblickstabelle: Technik vs. Filmsprache
- Glossar mit 55 Fachbegriffen (suchbar, filterbar)
- Quellenverzeichnis mit 70 verifizierten Quellen (Faktencheck März 2026)
- Vollständig zweisprachig: Deutsch / Englisch

## Tech Stack

- **Frontend:** React 19 + TypeScript
- **Styling:** Tailwind CSS 4 + shadcn/ui
- **Animationen:** Framer Motion
- **Build:** Vite
- **Fonts:** Cormorant Garamond (Display) + Lato (Body) + JetBrains Mono (Daten)

## Design

"Dunkles Lichtspiel" – Dark Editorial Cinema. Dunkelheit als Bühne, Licht als Erzähler. Gold/Bernstein als Akzent, Dunkelrot für Gegenreaktionen. Vertikale sticky Timeline-Navigation links, Inhalte rechts.

## Entwicklung

```bash
pnpm install
pnpm dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.
